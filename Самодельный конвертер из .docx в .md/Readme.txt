Самодельный конвертер из .docx в .md.

*** Написан с помощью библиотеки python-docx.

Весь код разделён на 3 файла - main.py, main_menu.py и converter.py для более удобного и читабельного кода.
+ Есть готовая версия, скомпиленная в .exe-шник.
Запускаем, выбираем docx, говорим куда надо сохранить и сохраняем уже в md.

В файле main.py лежит функционал открытия окна проводника Windows,
выбора файла, сохранения, пути для сохранения и т.д.
Этот файл также как и главное меню является исполняемым, его можно запустить отдельно, без главного меню и
работать с конвертером без главного меню, хотя это не удобно.

В файле main_menu.py реализовано главное меню, которое по-сути и является главным исполняемым файлом, 
через который и осуществляется запуск всей программы по цепочке, т.е. main_menu.py <---> main.py <---> converter.py
+ Есть возможность выхода из программы по нажатию на 0.

В файле converter.py лежит функционал самой конвертации из docx в md.
В нём заведена функция с двумя аргументами для входящего и исходящего путей.
Дальше создаётся объект, который загружает указанный вордовский файл для дальнейшего чтения.
После этого создаётся пустой список строк Markdown.
После этого включаем цикл for, который перебирает каждый параграф в документе.
Т.е. по-сути парсим документ, убирая лишние пробелы и проверяя, есть ли в файле пустые параграфы и заголовки.
После всех проверок, открываем файл на запись (output_path) с кодировкой utf-8 и записываем в него
все строки из md_lines, соединённые \n, т.е. с переносом строки.

Т.е. идея в том, чтобы спарсить документ docx, произвести с ним некие манипуляции и записать в другой документ с расширением md, всё.
